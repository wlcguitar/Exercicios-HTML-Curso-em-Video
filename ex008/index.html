Principais Formatações.

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântico).

Nesta frase temos um termo em destaque usando a tag <strong> (semântico)

Itálico / Enfânse

Nesta frase, temos um termo em intálico usando a tag <i> (não semântico)

Nesta frase, temos um termo em enfâse usando a tag <em> (semântico).

Texto Marcado.

Podemos criar também um texto marcado usando a tag <MARK>

E no outro paragrafo temos outro texto marcado.

Texto Grade e Pequeno.

Estamos criando um texto grande e um texto pequeno.

Texto deletado.

Podemos marcar um texto como deletado ele deve ser lido mais não considerado, usando a tag <del>.

Texto inserido.

Podemos marcar um texto como inserido para dar enfâse ao indicar que ele foi inserido depois, usando a tag <ins>.

Sublinhado

Podemos também sublinahr um texto usando a tag <u> (semântica).

Texto sobrescrito.

Para inserir coisas do tipo x 20+ 3, usando a tag <sup>.

Texto subscrito.

Para inserir coisas do tipo h2o, usando a tag <sub>.
